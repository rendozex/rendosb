Docker / Ollama Setup

                  
Run a local model in Docker, then pick it from the Model dropdown above.

                  
    
                    
  1. Install Docker Desktop and start it.
    2. 
                    
  2. Open PowerShell and run:
                      
    docker run -d --name ollama -p 11434:11434 ollama/ollama
    
                    
    3. 
                    
  3. Pull a model:
                      
    docker exec -it ollama ollama pull llama3.2
    
                    
    4. 
                    
  4. Confirm Ollama is running — you should see models in the Model dropdown.
    5. 
                  

                  
Default URL: http://127.0.0.1:11434 (change in Settings if needed)
